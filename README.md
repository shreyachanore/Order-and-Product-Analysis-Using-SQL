# Order-and-Product-Analysis-Using-SQL

## About Project
This is a beginner-level SQL project focused on analyzing order and product data.
The data is stored in CSV files and analyzed using SQL queries to understand sales and product performance.

## Tools Used
- SQL (MySQL)
- Excel (CSV files)
- GitHub

## Dataset
The project uses the following datasets:
- Customers.csv
- Orders.csv
- Orderitem.csv
- Products.csv

## What I Did
- Imported CSV files into SQL tables
- Used SELECT queries to explore the data
- Used JOIN queries to combine order and product information
- Used basic aggregation functions like COUNT and SUM for analysis

## Files in This Repository
- sql_queries.sql : Contains all SQL queries used in the project
- CSV files : Dataset used for analysis

## Conclusion
This project helped me understand how SQL is used for basic data analysis on real datasets.
It improved my skills in writing queries and working with multiple tables.
